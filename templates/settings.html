Need Help?

How to find the system tray icon:
  1. Look in the bottom-right corner of your screen (near the clock)
  2. Click the ^ arrow to show hidden icons
  3. Find the 💙$ icon (blue with dollar sign)
  4. Right-click it for menu options
How to quit the app:
  • Background Mode: Right-click system tray icon → Quit
  • Normal Mode: Just close the browser
  • Alternative: Open Task Manager (Ctrl+Shift+Esc) → End "ExpenseTracker.exe"
